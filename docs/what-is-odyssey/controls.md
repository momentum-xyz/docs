# Odyssey controls

In Odyssey, you use a combination of keyboard and mouse to move in the 3D space:

|Key|Function|
|---|---|
| `W` | Move forward |
| `A` | Move left|
| `S` | Move backwards |
| `D` | Move right |
| `E` | Move up |
| `Q` | Move down |
| `SHIFT` | Go faster |
| `LMC` | Left mouse click to select objects |
